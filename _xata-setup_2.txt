Xata :  Merupakan pihak ketiga yang dapat membantu dalam menyediakan sebuah database seperti Sekarang PostGres

    STEP BY STEP SETUP DB DAN INCLUDE KE PROGRAM 
        #   Xata DB : 
            1. Kunjungi :   https://lite.xata.io/
                - daftarkan akun anda dan buat database anda 
                - kemudian akan diberikan beberapa file .env pilih generate text maka akan diberikan file .txt 
                  lalu copy semua isinya.
                - Saat disuruh memilih orm gunakan drizzle ORM SQL TS
                - Step slanjutnya kita langsung klik finish karena kita akan menggunakan integrasi nya menggunakan 
                  xata cli.

        #   Xata Auth : 
            Xata bisa digunkan sebagai adapter authentikasi dimana kita akan menyimpan session baik data melalui xata
            untuk user yang login dan sign up 
            1. Installl Xata CLI :   npm i -g @xata.io/cli
            2. Jalankan comand berikut : 
                    - xata auth login
                        Q   : pada API Key pilih yang ada dan copy xata api pada .env
                    - xata init : 
                        Q   : pada db pilih yang anda setup di lite.xata 
                        Q   : gunakan bahasa di pakai misa TS 
                        Q   : enter maka dia akan membuat beberapa file 
            3. Setup : 
                - Pertama pindahkan xata.ts di dalam src ke root project kemudian src nya di delete: 
                - kemudian Perbaurui isinya sesuai dengan berikut :
